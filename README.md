# gm.xyz
### Installation
~~~
git clone https://github.com/ericet/gmXyz.git
cd gmXyz
npm install
~~~

### Setup
* mv .env.example .env
* Fill in your USERID and TOKEN 
  * USERID - Your ENS name or wallet address
  * TOKEN - press F12 and copy accessToken value from Application->Local Storage->https://gm.xyz


### Auto Following back
~~~
node autoFollow.js
~~~
